ℹ️ О проекте

AI Stack Guide — учебный веб-каталог ИИ-инструментов, созданный на основе сравнительного анализа 40+ сервисов.

🎯 Цель

Помочь студентам и специалистам из РФ быстро находить подходящие ИИ-инструменты с учётом:

  • ✅ Доступности без VPN
  • 🌐 Категории задач (текст, изображения, видео, код)
  • ⭐ Качества понимания русского языка

🔧 Технологии

  • HTML5 / CSS3 / Vanilla JavaScript (ES6+)
  • Адаптивная вёрстка (mobile-first)
  • Локальные данные (без бэкенда)
  • Виртуальный помощник с правило-базой ответов

📦 Данные

Информация об инструментах взята из учебного документа преподавателя. Проект носит образовательный характер.

🚀 Запуск

  1. Скачай папку проекта
  2. Открой index.html в браузере
  3. Готово! Сервер не требуется
